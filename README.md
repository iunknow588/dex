# 智运通交易系统 (LuckEngine)

智运通交易系统 (LuckEngine) 是一个基于 Injective 区块链构建的去中心化交易所（DEX），专注于合约交易，提供专业级的订单簿交易体验。系统提供三大核心交易模式：低风险合约交易、无风险套利交易、低风险闪电贷模式。

## 🚀 项目特色

- **去中心化**: 基于 Injective 链，无中心化风险
- **高性能**: 利用 Injective 链的高性能特性
- **专业交易**: 完整的订单簿模型
- **三大交易模式**: 低风险合约交易、无风险套利交易、低风险闪电贷模式
- **用户友好**: 简洁易用的界面设计
- **安全可靠**: 多层安全防护机制

## 🎯 三大交易模式

### 1. 低风险合约交易
- **动态对冲系统**: 基于价格监控的智能风险控制
- **互斥对冲单**: 同时只能存在一个对冲单（风险锁定或利润锁定）
- **用户控制**: 支持手动干预和自动执行模式
- **实时监控**: WebSocket 实时监听价格变化
- **风险等级**: 中等

### 2. 无风险套利交易
- **跨交易所扫描**: 自动检测多个交易所间的价格差异
- **智能执行**: 自动检测并执行价格差异套利机会
- **风险评估**: 实时计算套利风险（滑点、网络延迟等）
- **策略丰富**: 价格差异套利、三角套利、统计套利
- **风险等级**: 低

### 3. 低风险闪电贷模式
- **无本金交易**: 利用闪电贷进行无本金杠杆交易
- **策略模板**: 预置多种闪电贷交易策略模板
- **风险控制**: 多层次风险评估和自动化保护机制
- **原子性保证**: 单笔交易内完成借贷、交易、还款全流程
- **风险等级**: 高

## 🛠️ 技术栈

- **前端框架**: React 19.2.0 + TypeScript 5.9.3
- **构建工具**: Vite
- **UI 框架**: Material-UI
- **状态管理**: Zustand + React Query
- **路由管理**: React Router v6
- **区块链 SDK**: @injectivelabs/sdk-ts
- **部署平台**: Vercel

## ⚠️ 风险提示

### 通用风险提示

1. **区块链风险**: 智能合约和区块链存在固有风险
2. **市场风险**: 加密货币价格波动可能导致损失
3. **技术风险**: 网络异常、系统故障可能影响交易
4. **操作风险**: 操作失误可能导致意外损失

### 模式特定提醒

#### 合约交易提醒
- 设置合理的止损和止盈价格
- 关注保证金充足率
- 避免过度杠杆

#### 套利交易提醒
- 套利机会稍纵即逝，及时执行
- 注意Gas费用对收益的影响
- 市场异常波动时暂停套利

#### 闪电贷交易提醒
- 仔细阅读策略说明和风险提示
- 从小额开始测试策略效果
- 关注协议的临时维护和升级

### 安全最佳实践

1. **保护私钥**: 永远不要分享您的私钥或助记词
2. **谨慎授权**: 只连接可信任的应用
3. **风险意识**: 区块链交易存在较大风险，请谨慎操作
4. **备份数据**: 定期备份重要数据

**记住**: 理性投资，控制风险。加密货币交易存在较大风险，请务必了解相关风险后再进行操作。
